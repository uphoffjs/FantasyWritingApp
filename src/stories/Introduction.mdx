# FantasyWritingApp Component Library

Welcome to the FantasyWritingApp design system and component library. This documentation provides everything you need to build consistent, beautiful, and accessible interfaces.

## üìö What is this?

This Storybook contains:
- **Component Library**: All UI components used in the FantasyWritingApp
- **Design Tokens**: Colors, typography, spacing, and other design values
- **Patterns & Guidelines**: Best practices for common UI patterns
- **Interactive Playground**: Test components with different props and states

## üöÄ Quick Start

### For Developers

1. **Browse Components**: Use the sidebar to explore available components
2. **View Stories**: Each component has multiple stories showing different states
3. **Try Controls**: Modify props in real-time using the Controls panel
4. **Check Docs**: Read usage guidelines and API documentation
5. **Copy Code**: Use the "Show code" button to get implementation examples

### For Designers

1. **Review Visual Design**: See how designs translate to code
2. **Test Interactions**: Try hover states, clicks, and animations
3. **Check Responsive**: Use viewport addon to test different screen sizes
4. **Verify Accessibility**: A11y tab shows accessibility compliance

## üé® Design System

### Color Palette
Our fantasy-inspired color system with semantic naming:
- **Primary**: Mystical teal/green tones
- **Secondary**: Deep purple accents
- **Tertiary**: Warm amber highlights

### Typography
- **Headings**: Cinzel (fantasy aesthetic)
- **Body**: Crimson Text (readable serif)
- **UI**: System fonts for interface elements

### Spacing
8px grid system for consistent spacing across all components.

## üß© Component Categories

### Atoms
Basic building blocks like buttons, inputs, and icons.

### Molecules
Simple component groups like cards, form fields, and list items.

### Organisms
Complex components like headers, forms, and data tables.

### Pages
Full page layouts showing complete user interfaces.
